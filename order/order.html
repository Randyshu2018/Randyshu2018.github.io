# 订单相关API

# 创建订单 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法
describe.platformNo string 是 平台编号
describe.ebaasUserId string 是 ebaas会员号
data.orderNo string 是 订单编号
data.customerId string 是 客户会员号
data.merchantId string 是 商户号
data.digest string 是 摘要
data.signature string 是 签名
data.invoiceIsCreate string 是 是否需要开具发票 1[是(invoice.xxx信息必填)];0[否]
data.discounts Array 否 折扣(预留字段)
data.skus.skuNo Array 是 SKU编号
data.skus.goodsInfo string 是 商品信息
data.skus.skuNo Array 是 SKU编号
data.skus.goodsInfo string 是 商品信息
data.deliveryInfo.phone string 是 手机号
data.deliveryInfo.deliveryType number 是 运输方式(1:线上授权 2:线下邮件)
data.deliveryInfo.customerNo string 是 会员编号
data.deliveryInfo.customName string 是 客户名称
data.deliveryInfo.email string 是 邮箱地址
data.invoice.invoiceClassification string 是 发票类型
data.invoice.invoiceType string 是 开具类型
data.invoice.invoiceName string 否 发票抬头
data.invoice.invoiceNo string 否 发票号
data.invoice.taxRegisterCode string 否 税务登记证号
data.invoice.bankName string 否 开户银行名称
data.invoice.baseAccountNo string 否 基本开户账号
data.invoice.registerAddress string 否 注册场所地址
data.invoice.registerPhone string 否 注册固定电话
data.invoice.postalCode string 否 快递编号
data.invoice.receiveAddress string 否 发票邮寄地址

# 修改订单价格 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法:修改订单价格
describe.ebaasUserId string 是 ebaas会员号
data.orderNo string 是 订单编号
data.itemId string 是 订单条目编号
data.newAmount number 是 新价格
data.digest string 是 摘要
data.signature string 是 签名

# 使用示例

请求参数:

{
  "platformNo": "E008",
  "orgName": "SampleOrg",
  "enrollmentID": "local-randy",
  "password": "123456",
  "describe": {
    "channel": "ch1",
    "chaincode": "commercial_platform",
    "method": "Order:changeAmount",
    "ebaasUserId": "randy"
  },
  "data": {
    "orderNo": "ORDER100",
    "itemId": "1_ORDER100",
    "newAmount": 1000,
    "digest": "none",
    "signature": "none"
  }
}
