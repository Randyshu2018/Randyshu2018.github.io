# 发票相关API

# 创建发票 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法:创建发票
describe.platformNo string 是 平台编号
describe.ebaasUserId string 是 ebaas会员号
data.businessId string 是 业务编号
data.invoiceClassification string 是 发票类型
data.invoiceType string 是 开具类型
data.amount number 否 部分金额退款后重新开票使用
data.orderNo string 是 订单编号
data.invoiceName string 否 发票抬头
data.invoiceNo string 否 发票号
data.taxRegisterCode string 否 税务登记证号
data.bankName string 否 开户银行名称
data.baseAccountNo string 否 基本开户账号
data.registerAddress string 否 注册场所地址
data.registerPhone string 否 注册固定电话
data.postalCode string 否 快递编号
data.receiveAddress string 否 发票邮寄地址

# 使用示例

请求参数:

{
  "platformNo": "E008",
  "orgName": "SampleOrg",
  "enrollmentID": "local-randy",
  "password": "123456",
  "describe": {
    "channel": "ch1",
    "chaincode": "commercial_platform",
    "method": "Invoice:create",
    "platformNo": "E008",
    "ebaasUserId": "randy"
  },
  "data": {
    "businessId": "aaa",
    "invoiceClassification": "1",
    "invoiceType": "1",
    "amount": 12000,
    "orderNo": "ORDER100",
    "invoiceName": "添饥",
    "invoiceNo": "GWQRV",
    "taxRegisterCode": "123",
    "bankName": "中国银行",
    "baseAccountNo": "zgyh",
    "registerAddress": "上海静安区",
    "registerPhone": "17625495024",
    "postalCode": "aaa",
    "receiveAddress": "上海静安区"
  }
}

# 发票状态通过 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法:更改发票状态
describe.ebaasUserId string 是 ebaas会员号
data.invoiceRequestId string 是 发票申请编号
data.customerId string 是 用户id
data.invoiceNo string 是 发票号

{

    "channel": "ch1",

    "method": "Invoice:invokePass",
    "ebaasUserId": "randy"
  },
  "data": {
    "invoiceRequestId": "invoice_ORDER100",
    "customerId": "mjj",
    "invoiceNo": "GWQRV",
    "updatedAt": "1591891593274"
