# 分账相关API

# 创建分账策略 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法:创建分账策略
describe.id string 是 分账策略ID
describe.ebaasUserId string 是 ebaas会员号
data.partitionMap.merchant number 是 商户分账比例
data.partitionMap.platform number 是 平台分账比例

# 使用示例

请求参数:

{
  "platformNo": "E008",
  "orgName": "SampleOrg",
  "enrollmentID": "local-randy",
  "password": "123456",
  "describe": {
    "channel": "ch1",
    "chaincode": "commercial_platform",
    "method": "SubAmount:createPolicy",
    "id": "POLICY_1",
    "ebaasUserId": "randy"
  },
  "data": {
    "partitionMap": {
      "merchant": 0.99,
      "platform": 0.01
    }
  }
}

# 确认分账信息 post /api/invoke

# 参数

参数名 类型 是否必填 说明
platformNo string 是 平台编号
orgName string 是 组织名称
enrollmentID string 是 用户名
password string 是 密码
describe.channel string 是 通道名称
describe.chaincode string 是 合约名称
describe.method string 是 合约方法:确认分账信息
describe.id string 是 支付ID
describe.ebaasUserId string 是 ebaas会员号
data.subAmountNo string 是 分账条目ID
data.payeeSignature string 是 用户的签名
data.paidAmount number 是 支付金额

# 使用示例

请求参数:

{
  "platformNo": "E008",
  "orgName": "SampleOrg",
  "enrollmentID": "local-randy",
  "password": "123456",
  "describe": {
    "channel": "ch1",
    "chaincode": "commercial_platform",
    "method": "SubAmount:confirmItem",
    "id": "PAYMENT_2",
    "ebaasUserId": "randy"
  },
  "data": {
    "subAmountNo": "order_1_ORDER100_merchant_1FEC4401EE743EC96B1D343B1E847711",
    "payeeSignature": "PAYEE_SIGNATURE",
    "paidAmount": 990
  }
}
